# 📊 Sistema de Dashboard para Consultores

Este projeto é um **dashboard interativo** desenvolvido em **Streamlit** para gerenciar vendas, consultores e clientes, com relatórios e rankings automáticos.  
Ele utiliza **MongoDB** como banco de dados e foi estruturado de forma modular para facilitar manutenção e expansão.

---

## 🚀 Funcionalidades

- **Dashboard Inicial**: visão geral de vendas e comissões.
- **Cadastro de Consultores**: adicionar novos consultores ao sistema.
- **Gerenciar Vendas**:
  - Cadastrar vendas (com 13 parcelas de comissão, apenas a primeira marcada por padrão).
  - Editar ou excluir vendas existentes.
- **Buscar Cliente**: localizar vendas por cliente.
- **Buscar Consultor**: localizar vendas por consultor.
- **Comissão por Mês**: relatório detalhado de comissões mensais.
- **Rankings e Relatórios**:
  - Ranking acumulado.
  - Ranking por mês (com filtro).
  - Ranking por semana (com filtro).
  - Destaque para os **Top 3 consultores** em cada ranking.
